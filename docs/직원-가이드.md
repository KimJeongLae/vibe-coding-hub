# 직원용 가이드 — 내 페이지 만들기 🚀

바이브코딩 클래스에 오신 걸 환영해요! 아래 순서대로 따라 하면
**Claude Code로 코딩 → GitHub push → 자동 배포**까지 직접 경험하게 됩니다.

> 준비물: GitHub 초대 수락, 관리자에게 받은 Supabase 키 2줄

---

## 0. 필요한 것 설치 (한 번만)
- [Node.js](https://nodejs.org) (LTS 버전)
- [Git](https://git-scm.com)
- [Claude Code](https://claude.com/claude-code) 또는 VS Code

## 1. 프로젝트 내려받기
```bash
git clone https://github.com/KimJeongLae/vibe-coding-hub.git
cd vibe-coding-hub
npm install
```

## 2. 로컬에서 실행
```bash
npm run dev
```
브라우저에서 http://localhost:3000 → **본인 이름 셀** 클릭 → 내 페이지로!

## 3. ⭐ 내 페이지 만들기 (핵심!)
내 작업물은 **내 페이지 파일을 직접 코딩**해서 만듭니다. 폼에 입력하는 게 아니에요.

- 내 파일 위치: **`app/members/내slug/page.tsx`**
  (내 slug 는 주소창 `/members/여기` 부분 — 예: `hong-jiho`)
- 이 파일을 Claude Code 로 열고 자유롭게 만들면, 그 결과가 **셀 페이지에 그대로** 나타납니다.

Claude Code 에게 이렇게 부탁해보세요:
- *"이 페이지에 내 소개 카드랑, 내가 만든 것들을 카드 그리드로 보여줘"*
- *"배경에 그라데이션 넣고, 버튼 누르면 카운트 올라가는 위젯 만들어줘"*
- *"내가 따로 배포한 사이트를 이 페이지 안에 iframe 으로 띄워줘"*

## 4. (선택) Supabase 로 동적 기능 추가
방명록·좋아요·목록처럼 **데이터를 저장**하는 기능을 만들고 싶을 때만 사용해요.
`.env.local.example` 을 복사해 `.env.local` 을 만들고 관리자가 준 값 2줄을 넣습니다.
```
NEXT_PUBLIC_SUPABASE_URL=관리자가-준-값
NEXT_PUBLIC_SUPABASE_ANON_KEY=관리자가-준-값
```
새 테이블은 Supabase 대시보드 → **SQL Editor** 에서 만들 수 있어요.
반드시 **본인 이름 접두사**를 붙이세요:
```sql
create table if not exists hong_todos (
  id uuid primary key default gen_random_uuid(),
  content text not null,
  created_at timestamptz default now()
);
alter table hong_todos enable row level security;
create policy "open" on hong_todos for all using (true) with check (true);
```
> ⛔ **남의 테이블은 절대 건드리지 마세요** (DROP/ALTER 금지).
> 같은 프로젝트라서, 만든 테이블은 코드에서 `supabase.from("hong_todos")` 로 **바로** 쓸 수 있어요.

## 5. 내 작업 올리기 (배포!)
```bash
git add .
git commit -m "홍지호: 내 페이지 꾸미기"
git pull --rebase   # 다른 사람이 먼저 올렸을 수 있으니 먼저 당겨오기
git push
```
push 하면 **Vercel이 자동으로 재배포**합니다.
잠시 뒤 라이브 사이트 https://vibe-coding-hub-kappa.vercel.app 에서 내 결과 확인! 🎉

## 자주 나는 문제
| 증상 | 해결 |
|---|---|
| 내 페이지가 안 바뀜 | 파일 저장 확인 후 `npm run dev` 재시작 (또는 브라우저 새로고침) |
| `git push` 가 거부됨 (rejected) | `git pull --rebase` 먼저 실행 후 다시 push |
| push 했는데 사이트에 반영 안 됨 | Vercel 배포가 끝날 때까지 1~2분 기다린 후 새로고침 |
| 셀에 내 이름이 없음 | 관리자에게 `lib/members.ts` 에 추가 요청 |
| (Supabase 쓸 때) "미연결" 표시 | `.env.local` 값 확인 후 `npm run dev` 재시작 |
